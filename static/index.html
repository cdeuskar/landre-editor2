LandRe — User Guide

What is LandRe?

LandRe is a planning tool designed to help you quickly create and adjust land readjustment layouts. It lets you experiment with different configurations and explore options before committing to a design — making it easier to facilitate discussions with stakeholders and test ideas in real time.

Important: The outputs produced by this app are intended for planning exploration and discussion only. They are not official site plans or survey-grade documents.

Getting started

Begin by loading the original parcels for your project area. You have two options:

  • Upload KML — import a KML file containing the original parcel boundaries.
  • Trace parcels — draw parcels directly on the satellite map by clicking to place vertices. Click the first vertex again (or press Done tracing) to close each polygon.

Once loaded, original parcels appear on the map as a reference layer. Use Edit originals if you need to adjust any of the source boundaries.

Readjusting parcels

Use the tools in the left toolbar to build your new layout:

  • Select / Reshape — click a parcel to select it, then drag its vertices or edges to reshape it. Hold Shift while dragging an edge to keep it perpendicular. Shared vertices between adjacent parcels are highlighted in purple; double-click a shared vertex or edge to drag it together with all other shared vertices and edges simultaneously, keeping boundaries aligned.
  • Draw polygon — draw a new parcel from scratch. Vertices snap to existing corners and road intersections automatically.
  • Draw road — trace a road centreline; the app generates the road polygon from a configurable width.
  • Delete polygon — click a parcel to remove it.
  • Resize selected parcels — type a percentage and click the button to shrink or grow selected parcels uniformly.
  • Clip overlaps — automatically resolves overlapping parcels (smaller parcels take priority) and trims any parcel area that overlaps a road.
  • Clip spillover — trims any parcel area that falls outside the project boundary.

Use Undo / Redo (or Ctrl+Z / Ctrl+Y) at any time to step back through your changes.

Understanding the statistics panel

The right panel shows a table of all parcels with key metrics updated in real time as you edit:

  • Δ% — how much each parcel has grown or shrunk relative to its original area. A negative value means the parcel is smaller than before.
  • Overlap w/ orig — what percentage of the new parcel footprint coincides with its original location.
  • Spillover — area that falls outside the project boundary.
  • Violations — the total count of parcels that exceed the configured thresholds (maximum reduction, minimum overlap, minimum size). When this reaches zero, the row turns green.

You can sort the table by any column and adjust the threshold parameters at the top of the panel.

Tip — Δ% colour gradient: Turn on the Δ% color gradient button in the toolbar to colour private parcels from red (most reduced) to green (least reduced). This gives you an immediate visual read of which parcels are carrying a disproportionate share of the land contribution — those in red are the ones most in need of adjustment to equalize the reduction across owners.

Exporting your work

  • Export KML — saves the current parcel layout as a KML file you can open in Google Earth or other GIS tools.
  • Export CSV — saves the statistics table as a spreadsheet.
